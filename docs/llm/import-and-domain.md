# Import & domain rules

This file should contain the rules relevant for the import and export process including conversion, but not the json format itself:

- **JSON import file format** — every rule about the structure and content of a
  `.json` import file (envelope, words, triples, formulas, sources, values,
  components, plus the `import_mapper` / `$dto` discipline) is now in
  `docs/llm/json_structure.md`.
- **Domain noun definitions** — what *word*, *triple*, *phrase*, *term*,
  *value*, etc. mean precisely — remain in `docs/llm/architecture.md`.
