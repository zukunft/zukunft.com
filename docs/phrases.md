Phrases
-------

A phrase is either a word or a triple

There are four levels to assign functionality to a phrase

1. simply be using the name in a formula. 
2. some single words or triples are used in the code, e.g. for the configuration. These phrases are admin protected and are selected by the standard name.
3. some single phrases are using a coded functionality. These phrases have a code_id
4. if coded functionality should be assigned to a group of phrases, the phrase type is used.
